An independent project should be accepted if it

(A) ** produces a net present value that is greater than or equal to zero.

(B) produces a net present value that is greater than the equivalent IRR.

(C) has only one sign reversal.

(D) produces a profitability index greater than or equal to zero.

EXPLANATIONS BELOW

Concept note-1: -According to the net present value theory, investing in something that has a net present value greater than zero should logically increase a company’s earnings. In the case of an investor, the investment should increase the shareholder’s wealth.

Concept note-2: -If the NPV is greater than $0, the project is accepted. Otherwise the project is rejected. The NPV is defined by its: r, which is the discount rate per period.

Concept note-3: -An acceptable project should have a net present value greater than or equal to zero and a profitability index greater than or equal to one. Whenever the internal rate of return on a project equals that project’s required rate of return, the net present value equals zero.

Concept note-4: -The acceptance rule for independent projects is to accept all projects where the IRR is above the required return (hurdle rate) for those projects. If projects are mutually exclusive, accept the one with the highest IRR (assuming it is above the hurdle rate).

Concept note-5: -Independent projects: If NPV is greater than $0, accept the project. Mutually exclusive projects: If the NPV of one project is greater than the NPV of the other project, accept the project with the higher NPV. If both projects have a negative NPV, reject both projects.
