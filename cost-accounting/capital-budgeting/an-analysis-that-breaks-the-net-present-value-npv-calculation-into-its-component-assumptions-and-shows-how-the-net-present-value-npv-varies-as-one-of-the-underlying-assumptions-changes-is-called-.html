An analysis that breaks the net present value (NPV) calculation into its component assumptions and shows how the net present value (NPV) varies as one of the underlying assumptions changes is called ____

(A) scenario analysis

(B) internal rate of return (IRR) analysis

(C) accounting break-even analysis

(D) ** sensitivity analysis

EXPLANATIONS BELOW

Concept note-1: -Scenario analysis breaks the net present value (NPV) calculation into its component assumptions and shows how the net present value (NPV) varies as each one of the underlying assumptions changes.

Concept note-2: -C) Scenario analysis considers the effect on net present value (NPV) of changing multiple project parameters.

Concept note-3: -Answer and Explanation: The answer is D) sensitivity analysis. Apart from scenario analysis, sensitivity analysis would test how NPV changes when one of related variables either increases or decreases.
